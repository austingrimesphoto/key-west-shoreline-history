# Contributing

Contributions are welcome, especially corrections from local historians, archivists, and GIS practitioners.

A geometry contribution must include:

- a permanent source URL or archival identifier;
- source date;
- rights or access notes;
- evidence class and confidence;
- a description of georeferencing or derivation;
- valid GeoJSON;
- a passing `python3 scripts/validate_geojson.py` run.

Do not submit hand-drawn historical coastlines without documentary evidence.
